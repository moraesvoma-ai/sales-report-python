# Relatório de Vendas - Nordeste Natural Açaí

Projeto simples em Python para análise automática de vendas de uma loja de açaí.

## Funções do programa

- Ler arquivo CSV de vendas
- Calcular faturamento do dia
- Contar vendas por produto
- Identificar produto mais vendido

## Tecnologias

- Python
- CSV

## Como executar

1. Baixe o repositório
2. Execute:

python main.py
